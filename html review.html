HTML Keyword Description
XML eXtensible Markup language, tool that describes and structures data on the internet
DTD Document type definition, defines tags and attributes used in XML/HTML
XSLT extensible stylesheet language transformations, used to transform a structured markup language to another form
HTML hypertext markup language uses tags and attributes to achieve font, color, graphic on the world wide web
CSS cascading style sheet, used to format the layout of web pages, define text styles, table sizes, font color, and more
Web Forms also known as HTML form, an online page that allows for user input
JavaScript object-oriented language used to create effects within web browsers
DOM document object model, platform that allows scripts to access and update content, structure, and style
DHTML dynamic html, creates interactive websites, by using a combination of HTML, Javascript, and CSS
PHP hypertext preprocessor, used to develop static websites, dynamic websites, or web applications
ASP & ASP.NET asp stands for active server pages both is a development framework for building web pages one was released in 1998 and one in 2002
JSP java server pages, technology that help software developers create generated web pages based on HTML, XML, and other document types
